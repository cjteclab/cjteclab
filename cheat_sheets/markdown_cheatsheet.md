# Cheatsheet markdown

## Escape Character
	- \
  
## Linebreaks
	- new line: use [double space] at the end of a line to format a new line
	- new paragraph: use [double enter] to get a new paragraph
	- listelements starts always in a new line

# Headers
	- Header_1: #
	- Header_2: ##
	- Header_3: ###
	- Header_4: #### 
	- Header_5: #####
	- Header_6: ######
	
	- with underscore: in next line === or ---

# Emphasis
	- italics: * [text] * 
	- bold: ** [text] **
	- strikethrough: ~~ [text] ~~

# Lists
	- ordered list: 1, 2, 3, ...
	- unordered list: - 

# Links
	- inline link: [text](link)
	- inline link with title: [text](link "title")
	- reference link: [text][link] with [link]: link

# Images
	- inline: ![alt text](link)
	- reference: ![alt text][link] with [link]: link

# Code Highlighting
	- inline: ` [text] `
	- block: ``` [text] ``

## Footnotes
	- inline: [^ [number] ] with [^ [number] ]: [text]

## Blockquotes
	- each line of the blockquote begins with >

## Horizontal Line
	- with hyphens: ---
